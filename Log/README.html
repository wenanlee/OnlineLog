OnlineLog

简单在线日志平台.

安装

测试平台:

Python 3.x

Windows 10 or Ubuntu server 18.04

运行

  • 本地运行
#默认模式运行 端口80
cd Log
python server.py
#自指定端口运行 端口 8080
cd Log
python server -p 8080
  • Docker

git clone https://github.com/wenanlee/OnlineLog.git
cd OnlineLog/
docker build -t log .
docker run -d -v $(pwd)/Log:/data --name Log -p 8080:80 log
